# Автоматизация мониторинга сельхозугодий с помощью БАС

Проект для демонстрации цифрового контура мониторинга сельскохозяйственных угодий с использованием беспилотных авиационных систем (БАС): от построения маршрутов облёта до подготовки агрорекомендаций на основе больших данных.


## Назначение

Скрипт выполняет три задачи:

1. Строит маршрут БАС по контуру поля в виде галсового обхода.
2. Экспортирует полетное задание в формате `QGroundControl .plan`.
3. Формирует агрорекомендации по CSV-наблюдениям.

## Файлы

- `bas_uav_monitoring.py` — основной скрипт
- `field_example.json` — пример контура поля
- `observations_example.csv` — пример входных агроданных

## Быстрый запуск

```bash
python bas_uav_monitoring.py \
  --field field_example.json \
  --observations observations_example.csv \
  --output-dir results \
  --altitude 60 \
  --camera-fov 78 \
  --side-overlap 0.30 \
  --route-angle 0 \
  --margin 4 \
  --cruise-speed 12
```

## Результаты

После выполнения в каталоге `results/` будут созданы:

- `mission.plan`
- `route_waypoints.csv`
- `agro_recommendations.csv`
- `summary.json`

## Что можно доработать дальше

- подключить реальные shapefile/geojson-контуры;
- строить маршрут с учетом запретных зон;
- добавить обработку ортофотопланов и индексов NDVI;
- интегрировать прогнозную модель урожайности;
- подключить БД и веб-интерфейс.
